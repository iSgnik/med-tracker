## 📌 Visão Geral e Problema Resolvido
O **MedTracker** é uma aplicação de linha de comando (CLI) simples e direta construída em Python. 
**A Dor Real:** Cuidadores familiares e idosos frequentemente se confundem com horários e listas de medicamentos, o que pode gerar superdosagem ou omissão de tratamentos essenciais.
**A Solução:** Uma ferramenta de terminal rápida onde o usuário pode registrar os remédios e listar a agenda diária de medicação de forma clara, funcionando como um bloco de notas virtual seguro e específico para saúde.

## 👥 Público-Alvo
Cuidadores de idosos, pessoas em tratamentos contínuos e qualquer indivíduo que precise organizar sua rotina medicamentosa básica.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3
* **Armazenamento:** Arquivo JSON local
* **Testes:** Pytest
* **Linting / Análise Estática:** Flake8
* **CI/CD:** GitHub Actions
